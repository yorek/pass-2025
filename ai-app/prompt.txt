DAB

I want to expose the stored procedure "dbo.SearchSimilarGrants" as a REST endpoint using Data API Builder. The endpoint must be named "GrantSearch". The database is SQL Server and the connection string is in the environment variable MSSQL. The stored procedure has a parameter named "SearchText". The endpoint connected to the stored procedure should answer to GET request and support QUERY graphql command. Also make sure to configure DAB to allow any CORS origin. The configuration file must be named dab-config.json. Use DAB CLI to do everything so that you can use the latest and greatest version of DAB.
